The hover effect can be done simply by using :hover in CSS.
However, :hover has limits on it's own. Thankfully, d3 can also
accomplish the same effects on it's own.

.on("mouseover", function(){

d3.select(this)

.attr("fill","gray");

});

When the mouse moves over a rect, the event listener for
that rect is triggered, the rect is selected as this, and
then the fill is set to gray. All of this is done without
CSS.
